# Favicons

## Generate favicons

There are many options, but one that can be helpful is [realfavicongenerator.net](https://realfavicongenerator.net/).

## Updates

To update favicons, update these files:

- `src/assets/favicon/`
- `static/favicon.ico`
- `src/routes/site.webmanifest/+page.ts`
- `src/routes/browserconfig.xml/+page.ts`
